# dockDesafio
Desafio Dock

## Tecnologias

- Node
- Jest (para os testes)

## Execução

Para instalar as dependências do projeto execute o comando:
```shell
npm i
```

Rodando a aplicação em produção:
```shell
npm start
```

Rodando a aplicação em dev:
```shell
npm run dev
```

## Testes

```shell
npm run test
```
